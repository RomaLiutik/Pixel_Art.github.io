Проект Pixel Art.(Активен)

Проект Pixel Art это редактор пиксельных изображений в формате 2D.
Идея для проекта была создана при неудобстве рисовать пиксельные
2D рисунки на бумаге.
Информация о проекте:
Язык программирования : python
Библиотека для компиляции в exe : pyinstaller
Библиотека для создания GUI : Tkinter

Pixel Art v0.1a

Первая версия проекта Pixel Art является тестовой версией
определения координат клика на холсте(для тех кто рабирается).
Информация о компонентах данной версии :
Work Space v1 :
Имеет размер 15х15 клеток.
Не изменяется.
Palette v1
Имеет два цвета :
Белый, чёрный

Установка :
Для того что-бы установить готовое приложение exe
